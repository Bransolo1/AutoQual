# Data Processing Addendum (Template)

This template is provided for enterprise onboarding and can be adapted per customer.

## Parties
- Controller: Customer
- Processor: Sensehub Auto Qual (Operator)

## Scope
Processing of research data, media, transcripts, and operational logs required to deliver
the services described in the master agreement.

## Processing details
- Categories of data subjects: researchers, participants, client stakeholders
- Types of data: identifiers, study metadata, media, transcripts, insights, audit logs
- Duration: per contract terms and retention settings

## Security measures
- Role-based access control and workspace isolation
- Audit logging for sensitive actions
- Encryption in transit
- Access revocation and token rotation

## Subprocessors
See `docs/trust-center/SUBPROCESSORS.md`.

## Data transfers
Cross-border transfers are governed by the agreement and applicable SCCs if required.

## Data subject requests
The operator provides tools to export and delete data per agreed procedures.
